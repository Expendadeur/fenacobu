# fenacobu
FENACOBU (Fédération Nationale des Coopératives d'Épargne et de Crédit du Burundi) est une plateforme web complète de gestion des opérations bancaires de guichet. Ce système permet aux institutions de microfinance et coopératives d'épargne et de crédit de gérer efficacement leurs opérations quotidiennes.
